QGIS Maps

July 19, 2024

I stumbled upon the book Discover QGIS 3.X - Second Edition by Kurt Menke and decided to go through it. I enjoy learning through books so I thought this would be a good way to get more acquanted with QGIS and strengthen some GIS fundamentals at the same time. These are some of the projects from the book.

Spatial Analysis

  • Filtered layers and selected features using SQL expressions
  • Used Dissolve, Buffer, and Clip processing algorithms
  • Used buffer operation combined with select by location operation to determine roads and parcels affected by a tornato
  • Used basic statistical tool to determine total number of parcels impacted
  • Conducted a site selection analysis to determine acceptable airports for a given set of constraints
  • Used the Graphical Modeler to build a geoprocessing model for the workflow
  • Utilized network routing and allocation analysis
  • Compute network isochrones using he QNEAT3 plugin
