Topo Map

This project was to see if I could create a topographic map or trail map totally within a desktop GIS. I utilized OpenSteetMaps data for map features and a LiDAR derived elevation model from the USGS for the hillshade. Using the built in blending tool was a little more difficult than GIMP or Photoshop, but its pretty close.

Tools: QGIS
Data Sources: USGS and OpenSteetMaps
