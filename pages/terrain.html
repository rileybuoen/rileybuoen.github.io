Terrain Map: The San Gabriel Mountains

This was the final project for my Advanced Cartography class at Humboldt State University, now Cal Poly Humboldt, in 2019. I used the techniques I leaned throughout this class to create the shaded relief and enhance it with Photoshop, blend the relief with overlaying raster datasets, processing and simplifying vector data, manage label placemnt following best practices, and combining it together in Illustrator. I took this map to the California Geographic Society conference and it won the 2nd place Student Paper Map Award in 2019. Looking back at this now I notice there is a lot I would do different, so that is why I am activly working on a revised version of this map with those changes.

Programs: ArcGIS, Photoshop, Illustrator
Data: USGS, NLCD, GNIS
