Fire Station Response Times

This project utilized GRASS tools to create the road network using the fire station points and the road layers. Then using the QNEAT 3 plugin in QGIS, I computed the isocrone using the network and fire station layers. The QNEAT 3 plugin outputs a raster layer visualizing the distance in time from the input points. I then styled and blended that into the road network to make the network look like its glowing the closer to a fire station you are and falling into the background if you were farther away. Finally, I composed the final map in the map composer and finished it in GIMP.

Tools: QGIS, GIMP
Data: City of San Diego
