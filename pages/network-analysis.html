Fire Station Response Times

This project utilized GRASS tools for creating the network between the fire station and the road network layers. Then using the QNEAT3 plugin, I compute the isocrone from the location of the fire stations. QNEAT 3 outputs a raster layer which I then styled and blended into the road network to make the network look like its glowing. Finally, I composed the final map and finished it in GIMP.

Tools: QGIS, GIMP
Data: City of San Diego
