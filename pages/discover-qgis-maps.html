Discover QGIS 3.X Maps

I recently stumbled upon the book Discover QGIS 3.X - Second Edition by Kurt Menke and decided to pick up a copy and go through it. I enjoy learning through books so I thought this would be a good way to get more acquanted with QGIS and strengthen my GIS fundamentals at the same time. These are some of the projects in the book.

  • Filtered layers and selected features using SQL expressions
  • Used Dissolve, Buffer, and Clip processing algorithms
  • Preformed join between table and attribute table
  • Classified data using the Graduated renderer on numeric attributes
