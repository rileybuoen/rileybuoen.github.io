Discover QGIS 3.X Maps

I recently stumbled upon the book Discover QGIS 3.X - Second Edition by Kurt Menke and decided to pick up a copy and go through it. I enjoy learning through books so I thought this would be a good way to get more acquanted with QGIS and strengthen my GIS fundamentals at the same time. These are some of the projects from the book.

Spatial Analysis

  • Filtered layers and selected features using SQL expressions
  • Used Dissolve, Buffer, and Clip processing algorithms
  • Preformed join between table and attribute table
  • Classified data using the Graduated renderer on numeric attributes
  • Used buffer operation combined with select by location operation to determine roads and parcels affected by a tornato
  • Used basic statistical tool to determine total number of parcels impacted
